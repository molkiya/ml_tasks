# ITOIB - Учебный проект

Этот репозиторий содержит наработки и выполненные учебные задачи по машинному обучению и обработке данных.

## Описание

Проект представляет собой коллекцию различных заданий и экспериментов, выполненных в рамках учебного курса. Здесь собраны реализации различных алгоритмов машинного обучения, методы обработки данных и визуализации результатов.

## Структура проекта

```
ITOIB/
├── scripts/                    # Основные скрипты с учебными задачами

│   ├── fraud.py                # Обнаружение мошенничества с кредитными картами
│   ├── LDA.py                  # Тематическое моделирование и классификация с LDA
│   ├── pca_classification.py   # Классификация с использованием PCA и SVD
│   ├── tree_classification.py  # Классификация с деревом решений
│   ├── single_perceptron_regression.py  # Регрессия с однослойным перцептроном
│   └── metrics_visualizer.py   # Визуализация метрик моделей
│
├── utils/                      # Вспомогательные утилиты
│   ├── affinity_propagation.py
│   ├── decision_tree.py
```

## Установка и настройка

1. Клонируйте репозиторий или скачайте проект
2. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   source venv/bin/activate  # для Linux/Mac
   # или
   venv\Scripts\activate  # для Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

Все основные скрипты находятся в папке `scripts/`. Для запуска скрипта используйте:

```bash
python scripts/название_скрипта.py
```

### Примеры запуска:

- **Кластеризация**: `python scripts/affinity_propagation_clusterization.py`
- **Обнаружение мошенничества**: `python scripts/fraud.py`
- **LDA классификация**: `python scripts/LDA.py`
- **PCA классификация**: `python scripts/pca_classification.py`
- **Дерево решений**: `python scripts/tree_classification.py`
- **Регрессия перцептроном**: `python scripts/single_perceptron_regression.py`

## Содержание задач

### 1. Кластеризация (Affinity Propagation)
- Кластеризация данных ирисов с добавлением шума
- Визуализация результатов кластеризации

### 2. Обнаружение мошенничества
- Анализ транзакций по кредитным картам
- Классификация с использованием GaussianNB и DecisionTree
- Построение ROC-кривых и анализ метрик

### 3. Тематическое моделирование (LDA)
- Обработка текстовых данных IMDB
- Обучение моделей Word2Vec (CBOW, Skip-gram)
- Классификация с использованием LDA

### 4. Классификация с PCA
- Применение PCA и Truncated SVD для снижения размерности
- Сравнение моделей с и без сжатия признаков

### 5. Дерево решений
- Реализация собственного алгоритма дерева решений
- Классификация на данных ирисов

### 6. Регрессия перцептроном
- Прогнозирование временных рядов
- Использование различных функций активации (tanh, sigmoid)

## Примечания

- Это учебный проект, содержащий наработки и выполненные задания

- Некоторые скрипты требуют наличия данных в соответствующих папках `data/`
